## 3. 버튼 스타일

| 버튼 | 스타일 |
|------|--------|
| 검토 시작 (메인 CTA) | 파랑, `rounded-lg`, 전체 너비, 비활성 시 gray-700 |
| Excel 다운로드 | 초록, `rounded-md`, 아이콘 포함 |
| 탭 (도면검토/설정) | 선택 시 `bg-blue-600`, 미선택 시 투명 + hover gray-700 |
| ROI 파일 선택 | 점선 테두리 `border-dashed`, hover 시 파란 테두리 |
| 파일 제거 (✕) | 텍스트 버튼, hover 시 red-400 |
| 분석 취소 | 텍스트 버튼 red-400 |
| 엔진 선택 카드 | `border-2 rounded-lg`, 선택 시 파랑/보라 테두리 |

## 4. 레이아웃 구조

```
┌─────────────────────────────────────────┐
│  헤더 (고정, border-b)                   │
│  로고 + 탭 네비게이션 (도면검토 / 설정)   │
└─────────────────────────────────────────┘
│  메인 (max-w-5xl, 중앙 정렬, px-6 py-8) │
│                                          │
│  [업로드 폼 상태]                        │
│  ┌──────────────┬─────────────────────┐ │
│  │ 왼쪽: 파일   │ 오른쪽: 블록설정    │ │
│  │ - 도곽 DWG   │ - ROI 파일 선택     │ │
│  │ - 목록표 DWG │ - 개별 도면 블록명  │ │
│  │ - 개별 DWG   │ - 업로드 진행률     │ │
│  │              │ - 검토 시작 버튼    │ │
│  │              │ - 체크리스트        │ │
│  └──────────────┴─────────────────────┘ │
│                                          │
│  [Job 실행 상태]                         │
│  - 상태 헤더 + 취소 버튼                 │
│  - 진행률 바                             │
│  - 실시간 로그 패널 (h-56, 스크롤)       │
│  - 완료 시 다운로드 버튼                 │
└─────────────────────────────────────────┘
```

---

## 5. 컴포넌트 종류

| 컴포넌트 | 파일 | 설명 |
|---------|------|------|
| 헤더 | `App.tsx` | 로고 + 탭 네비게이션 |
| DropZone | `DropZone.tsx` | 드래그앤드롭 + 클릭 파일 선택, 파일 목록 표시 |
| BlockSelector | `BlockSelector.tsx` | ROI JSON 로컬 파일 선택, localStorage 캐시, 블록명 텍스트 입력 |
| JobProgress | `JobProgress.tsx` | 진행률 바 + SSE 실시간 로그 패널 + 취소 버튼 |
| DownloadButton | `DownloadButton.tsx` | Excel 파일 다운로드 링크 버튼 |
| SettingsTab | `SettingsTab.tsx` | 추출 엔진 선택 카드 (ezdxf / Vision AI) |
| 체크리스트 | `App.tsx` 인라인 | 제출 조건 충족 여부 표시 (○/✔) |
| 업로드 진행률 바 | `App.tsx` 인라인 | 파일 업로드 중 퍼센트 표시 |
